Hi, saya Muhammad Rizky Kurniawan


Saya dari Indonesia dan telah bekerja sebagai desainer UI UX selama lebih dari 5 tahun. Saya telah bekerja di perusahaan Belanda Frisseblikken sebagai desainer web dan pengembang front-end selama 3 tahun, saya membangun tim pengembangan web sendiri Duosweb sejak 2018, dan saat ini saya bekerja di Bold Australia sebagai Desainer UI sejak 2020. Saya sudah freelancing sejak 2012, dan saat ini saya menantikan untuk berkolaborasi dengan Anda!




Mengapa saya memulai dari youtube?

Pertama, saya ingin mengedukasi masyarakat di negara saya tentang Desain UI UX, karena pendidikan formal kami tidak mendapatkan banyak hal, terutama mereka yang tidak mampu membiayai kuliah. Makanya saya menggunakan bahasa Indonesia di video saya. Kedua, ini membantu saya mengembangkan merek pribadi saya, untuk mendapatkan lebih banyak keterpaparan kepada klien potensial sehingga mereka mengetahui cara saya berpikir dan cara saya mengomunikasikan pemikiran saya.
